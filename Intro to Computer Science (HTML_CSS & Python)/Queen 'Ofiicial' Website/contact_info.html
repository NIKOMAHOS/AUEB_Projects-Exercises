Αρχική Σελίδα Κατάστημα Προιόντα Επικοινωνία
HTML/ CSS Project 2021-2022

Η συγκεκριμένη εργασία έγινε στα πλαίσια του μαθήματος της Επιστήμης Ηλεκτρονικών Υπολογιστών, κατά την διάρκεια του πρώτου εξαμήνου και έτους 2021 - 2022.
Η εργασία αφορά την δημιουργία ενός ιστοτόπου, κυρίως(αλλά όχι μόνο) για αγορά υλικών αγαθών σχετικά με το βρετανικό ροκ συγκρότημα Queen.
Ο ιστότοπος περιέχει 5 κύριες σελίδες:
  • ΑΡΧΙΚΗ ΣΕΛΙΔΑ
  • ΚΑΤΑΣΤΗΜΑ
  • ΠΡΟΙΟΝΤΑ
  • ΕΠΙΚΟΙΝΩΝΙΑ
Και 6 ακόμη σελίδες, στις οποίες μπορεί ο επισκέπτης να πλοηγηθεί, οι οποίες αφορούν ειδικότερα την αγορά κάποιων συγκεκριμένων προιόντων:
  • Προιόν Μουσικής
  • Προιόν Βίντεο
  • Προιόν Ρουχισμού
  • Προιόν Συλλεκτιkού Αντικειμένου
  • Προιόν Αξεσουάρ
  • Προιόν Μάσκας
