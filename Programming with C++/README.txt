# FilmBrowser

- Για να πλοηγηθείτε στην συλλογή των ταινιών σύρετε το ποντίκι προσ την επιθυμητή κατεύθυνση κίνησης.
- Όταν φιλτράρονται οι ταινίες δεν αλλάζει η θέση τους, μόνο το αν φαίνονται ή όχι, αν πατήσετε κάποιο φίλτρο και "χαθούν" όλες οι ταινίες,
πλοηγηθείτε προς τις ταινίες προς το τέλος της λίστας.
- Αν θέλετε να επαναφέρετε τα φίλτρα πατήστε το κουμπί "Clear Filters".
- Η συλλογή με τις ταινίες περιέχει 10 συγκεκριμένες ταινίες.
